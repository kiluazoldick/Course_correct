## 🎯 Vision du Projet

**Corrige Tes Cours** est une plateforme éducative révolutionnaire qui utilise l'intelligence artificielle pour transformer la façon dont les étudiants apprennent. Notre mission est de rendre l'éducation plus accessible et efficace pour les étudiants universitaires camerounais grâce à des outils d'apprentissage intelligents et automatisés.

### 🌟 Ce que nous offrons

- ✨ **Résumés de cours automatiques** : L'IA analyse vos cours et génère des résumés clairs et structurés
- 🎓 **Quiz personnalisés intelligents** : Questions à choix multiples et ouvertes adaptées à votre contenu
- 💬 **Assistant IA "Tariq"** : Un chatbot éducatif motivant disponible 24/7
- 📊 **Suivi des performances** : Analyse de vos résultats et progression
- 📱 **100% Responsive** : Accessible sur mobile, tablette et ordinateur
- 🔒 **Sécurisé et fiable** : Authentification sécurisée et protection des données

## 📖 Guide d'Utilisation en Situation Réelle

### 🎬 Scénario 1 : Nouvel Utilisateur - Essai Sans Compte

**Situation :** Jean découvre Corrige Tes Cours et veut tester l'application avant de créer un compte.

#### Étape par Étape

1. **Accès à la plateforme**
   - Jean visite https://corrigetescours.com
   - Il voit la page d'accueil avec la section "Essayez maintenant"

2. **Upload anonyme pour tester**
   - Jean clique sur "Choisir un fichier" dans la section "Essayez maintenant"
   - Il sélectionne son cours de Mathématiques (PDF de 5MB)
   - Upload réussi → Redirection automatique vers la page de résumé

3. **Génération du résumé IA**
   - L'IA (DeepSeek R1) analyse automatiquement le cours
   - En 30 secondes, un résumé structuré apparaît
   - Jean lit le résumé et est impressionné par la qualité

4. **Conversion en utilisateur**
   - Jean clique sur "Télécharger le PDF"
   - Une modal de création de compte apparaît
   - Il crée son compte : jean@example.com / mot de passe
   - **Migration automatique** : Le cours testé est ajouté à son dashboard !

✅ **Résultat :** Jean a testé gratuitement, et son cours est déjà dans son compte. Aucune perte de données !

## 🔄 Système d'Onboarding Anonyme (Nouvelle Fonctionnalité !)

### 🎯 Objectif : Maximiser les Conversions

Notre système d'onboarding permet aux visiteurs de **tester l'application sans créer de compte**, puis convertit automatiquement leur essai en compte complet.

### ⚙️ Comment ça marche ?

#### 1. **Essai Anonyme (Sans compte)**
```
Visiteur → Upload fichier → Résumé IA généré → Voir le résultat
```
- Aucune inscription requise
- Génération de sessionId unique (localStorage)
- Résumé généré en temps réel
- Valide 48h (expiration automatique)

#### 2. **Conversion (Création de compte)**
```
Clic "Télécharger PDF" → Modal Signup → Compte créé → Migration automatique
```
- Le cours testé est **automatiquement transféré** dans le compte
- Aucune perte de données
- Expérience fluide et sans friction

#### 3. **Nettoyage Automatique**
```
Toutes les heures → Suppression des uploads expirés (>48h)
```
- Optimisation du stockage
- Respect de la vie privée
- Base de données propre

### 📊 Avantages Business

- ✅ **Taux de conversion augmenté** : Les utilisateurs testent avant de s'engager
- ✅ **Expérience sans friction** : Pas de barrière à l'entrée
- ✅ **Migration transparente** : Le cours testé devient le premier cours de l'utilisateur
- ✅ **Sécurité** : Expiration automatique après 48h

## 🚀 Installation & Configuration

### Prérequis
- Node.js 20+
- PostgreSQL (ou compte Neon)
- Compte OpenRouter (pour IA)
- Compte Lygos (pour paiements)
- Compte Google Cloud (pour OAuth, optionnel)

### Variables d'Environnement

Créer un fichier `.env` :

```env
# Database
DATABASE_URL=postgresql://user:password@host:5432/database
PGHOST=your-host
PGPORT=5432
PGUSER=your-user
PGPASSWORD=your-password
PGDATABASE=your-database

# Session
SESSION_SECRET=your-super-secret-session-key

# Google OAuth (optionnel)
GOOGLE_CLIENT_ID=your-google-client-id
GOOGLE_CLIENT_SECRET=your-google-client-secret

# AI (OpenRouter)
OPENROUTER_API_KEY=your-openrouter-api-key

# Payment (Lygos)
LYGOS_API_KEY=your-lygos-api-key

# Domain
CUSTOM_DOMAIN=https://corrigetescours.com
```

### Installation

```bash
# Cloner le projet
git clone <repository-url>
cd corrige-tes-cours

# Installer les dépendances
npm install

# Configurer la base de données
npm run db:push

# Lancer en développement
npm run dev
```

### Déploiement

L'application est configurée pour Replit avec :
- Port : 5000
- Host : 0.0.0.0
- Build Vite automatique en production
